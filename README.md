### 🔬 How I Approach Security Work

```
1. Collect     →  Capture logs / traffic / scan output (Splunk, Wireshark, Nmap)
2. Analyze     →  Look for anomalies, baseline deviations, known IOCs
3. Map         →  Tie findings to MITRE ATT&CK techniques
4. Respond     →  Follow structured IR steps (NIST framework)
5. Report      →  Document clearly for both technical & non-technical stakeholders
6. Improve     →  Feed learnings back into detection rules / baselines
```

This loop is the same structure I used across SentinelAI, the encrypted-traffic detection project, and the 5G slicing research — collect data, find the signal, map it to a known framework, and document it so someone else could act on it.
